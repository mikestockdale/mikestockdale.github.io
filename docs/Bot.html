1.1.4 Bot

Source code at bot.rkt

A bot is an entity that is controlled by client code. It can pick up a block entity. This is stored in the cargo field. It also has a list of nearby entities that it can see, in the neighbors field.
(struct bot (entity cargo neighbors) #:prefab)
 
(define (bot-id bot) (entity-id (bot-entity bot)))
(define (bot-location bot) (entity-location (bot-entity bot)))
We include a couple of helper functions, to access the entity id and location data.
